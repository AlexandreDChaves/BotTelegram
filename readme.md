# TechBot: Seu Bot Completo para Vagas e Cursos de Desenvolvimento!

Com o TechBot, você:

- **Encontra as melhores vagas de Dev**: Acesse uma lista atualizada de vagas em empresas renomadas, filtradas por suas habilidades e interesses.
- **Descobre cursos de alta qualidade**: Explore uma seleção de cursos online para aprimorar suas habilidades e se destacar no mercado.

## Tecnologias:

O TechBot é desenvolvido com as seguintes tecnologias:

- **Node.js**: Uma linguagem de programação leve e eficiente que permite a criação de aplicações robustas e escaláveis.
- **Telegraf**: Uma biblioteca JavaScript para criar bots interativos no Telegram, a plataforma de mensagens instantâneas mais popular do mundo.

## Como usar o TechBot:

### 1. Inicie o Bot:

- Abra o aplicativo Telegram no seu celular.
- Busque por "@TechBot" na barra de pesquisa.
- Clique no nome do bot e envie a mensagem "/start".

### 2. Explore as Opções:

- Uma apresentação amigável será exibida, apresentando as funcionalidades do TechBot.
- Escolha entre as opções "Vagas" ou "Cursos" pressionando o botão correspondente.

### 3. Encontre Vagas Relevantes:

- Selecione a área de desenvolvimento que te interessa (front-end, back-end, mobile, etc.).
- Filtre as vagas por localização, nível de experiência e outras qualificações.
- Acesse o link da vaga diretamente no Telegram para se candidatar.

### 4. Descubra Cursos Incríveis:

- Navegue por uma variedade de cursos online e presenciais de desenvolvimento.
- Explore diferentes áreas de conhecimento e níveis de aprofundamento.
- Acesse o link do curso no Telegram para se matricular.

### 5. Interaja com o Bot:

- Utilize o prompt do bot para digitar suas dúvidas, sugestões ou pedidos de pesquisa específicos.
- O TechBot está sempre pronto para te ajudar!
